Part of moving codes from MMO repository to separated repo and also running outside Unity.

Uses [Node.js](https://nodejs.org/), [NestJS](https://nestjs.com/), [Fastify](https://www.fastify.io/), and [Prisma](https://www.prisma.io/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
